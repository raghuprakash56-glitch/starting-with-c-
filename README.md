## 🚀 Getting Started

### Prerequisites
- GCC compiler or Clang
- Basic terminal knowledge

### Compiling and Running

**Compile a single program:**
```bash
gcc -o output_name program_name.c
./output_name
```

**With verbose warnings:**
```bash
gcc -Wall -Wextra -o output_name program_name.c
./output_name
```

### Example
```bash
cd introduction
gcc -o hello hello.c
./hello
```
